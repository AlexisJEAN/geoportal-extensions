Namespace: SearchEngine

L.geoportalControl.SearchEngine

Leaflet Control Class to search positons of geographic identifiers (places names, address, cadastral parcel) using :

  • the geocoding web service of the Geoportal Platform.
  • the autocompletion service of the Geoportal Platform

Use L.geoportalControl.SearchEngine() factory to create instances of that class.

Extends Leaflet L.Control native class.
